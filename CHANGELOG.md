# Changelog

All notable changes to ACF Panel Generator will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-02-06

### Added
- Initial release
- Component-based PHP file generation
- Recursive SCSS generation with nesting
- Support for all major ACF field types
- Configurable component field mapping
- Custom field templates
- Batch regeneration with overwrite mode
- Web-based interface
- Comprehensive configuration system
- Support for nested flexible content
- Support for repeater fields
- Support for group fields
- Support for relationship and post object fields
- Auto-generated SCSS index files
- Skip field configuration (by name and type)
- HTML wrapper customization

### Supported Field Types
- Text, Textarea, WYSIWYG
- Image, Gallery, File
- Link, URL
- Select, Radio, Checkbox, Button Group, True/False
- Number, Range
- Date Picker, Time Picker, Color Picker
- Relationship, Post Object
- oEmbed, Google Map
- Repeater (recursive)
- Flexible Content (recursive)
- Group
